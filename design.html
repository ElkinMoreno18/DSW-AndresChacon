Proposito del sitio

Este sitio esta dispuesto para hablar de como la pandemia hizo surgir la
educacion virtual, ya que con las necesidades presentadas por esta enfermedad
fue necesaria la implementacion para no dar pausa a la educacion que es lo mas importante.
Para los desarrolladores fue un punto de total importancia, ya que al tener que implementar objetos o medios de educacion digital se hizo mas importante el rol del desarrollador y tambien la creacion de sus herramientas para facilitar el aprendizaje por medios digitales como blogs o contenido multimedia y web.

En este sitio web se encontraran referencias y los diferentes aspectos del aprendizaje virtual que surgio a partir de la pandemia. Vamos a hacer una introduccion de las herramientas o mecanismos usados por los diferentes institutos educativos para realizar esta tarea de enseñanza por medios digitales.
